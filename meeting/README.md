# meeting/ — 면담 기록

강점 분석의 **1차 자료**가 들어가는 폴더입니다. 정기 1:1 면담·평가 기록을 보관합니다.

- **용도**: 동기·몰입·회피·자기인식을 파악하는 기준점
- **형식**: 인원별 면담 노트 (`*-meeting.md`)
- **주의**: 자기보고 기반이라 편향·시점 의존이 있고, 자기인식이 *학습된 방어(coping)*일 수 있음 → 다른 소스와 교차검증해 분별합니다(`../guide.md` 참고)

> 🔒 **개인정보.** 실제 면담 파일은 `.gitignore` 처리되어 로컬에만 보관됩니다. 이 README 외에는 커밋되지 않습니다.
