# workboard/ — 업무 이력

강점 분석의 **1차 자료**가 들어가는 폴더입니다. 도메인별 작업 이력을 보관합니다.

- **용도**: 누가 어떤 도메인에 익숙한지(**도메인 친숙도**) 신호를 추출
- **형식**: 작업 마스터 데이터 (`*.csv`)
- **주의**: 작업 "건수"는 **활동량이지 성과가 아닙니다**(미측정). 우열 판단 금지 — 램프업 비용 신호로만 사용합니다

> 🔒 **민감(담당자 실명·내부 링크 포함).** 실제 데이터 파일은 `.gitignore` 처리되어 로컬에만 보관됩니다. 이 README 외에는 커밋되지 않습니다.
