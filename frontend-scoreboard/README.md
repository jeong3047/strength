# frontend-scoreboard/ — 성과 스코어보드

팀의 목표·핵심결과(KR)·진척을 한눈에 보는 **스코어보드**와 구성원별 상세 보드가 들어가는 폴더입니다.

- **용도**: 강점 기반 배치안과 측정 지표를 시각화해 추적
- **형식**: 허브 페이지 + 구성원별 상세 페이지 (`*.html`)
- **원칙**: 배치는 확정이 아니라 **강점 기반 잠정안**이며, 계측 후 실적(KR)으로 재평가합니다
